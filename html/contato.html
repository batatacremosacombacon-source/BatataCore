Perguntas Frequentes

Como faço uma inscrição?

Basta acessar a página de inscrição, preencher os dados e confirmar o cadastro.

Posso cancelar minha inscrição?

Sim. Basta entrar em contato com nossa equipe antes da data do evento.

Os eventos são gratuitos?

Alguns eventos são gratuitos e outros possuem inscrição paga.

Os dados ficam seguros?

Sim. O BatataCore utiliza Supabase para armazenamento seguro dos dados.
